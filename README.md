# flexy
Practical flexbox examples for Vaasa WebDev Meetup

### Requirements

**Node.js** [http://nodejs.org/](http://nodejs.org) 

### How to use

```sh
npm i
gulp
```

### Browser support

FF, Chrome, IE Edge/11/10 (Windows 10)

FF and Chrome (Android 7)

### Needs improvement
